# 引用高亮功能说明

## 功能描述

在 Visualization 组件中点击任意两个参考文献之间的连线，会自动在 PaperHTML 组件中高亮显示所有包含这两篇参考文献引用的段落。

## 实现原理

### 1. 事件总线 (Event Bus)
- 使用 `mitt` 库创建全局事件总线
- 在 `main.js` 中初始化并通过 `provide` 注入到所有组件

### 2. Visualization 组件
- 为每条连线添加点击事件监听器
- 点击连线时发送 `highlight-references` 事件，携带 `from` 和 `to` 参数
- 点击时高亮当前连线（加粗、设置不透明度）
- 鼠标悬停时显示视觉反馈
- 提供"清除高亮"按钮

### 3. PaperHTML 组件
- 监听 `highlight-references` 事件
- 查找所有 `<sup>` 标签，匹配以下三种引用格式：
  - 单个引用：`[1]`
  - 范围引用：`[3-5]`
  - 多个引用：`[1,2,3]`
- 为匹配的段落添加 `.highlight-reference` 类
- 自动滚动到第一个高亮段落
- 监听 `clear-highlight` 事件以清除所有高亮

### 4. 视觉效果
- 高亮段落使用渐变背景色（黄橙色调）
- 添加左侧边框和阴影效果
- 使用动画实现脉冲效果，吸引注意力
- 平滑滚动到目标位置

## 使用方法

1. 在主页面中，点击任意连线
2. 文章区域会自动高亮相关引用并滚动到位置
3. 点击"清除高亮"按钮可清除所有高亮
4. 点击其他连线会自动切换高亮

## 技术栈

- Vue 3 Composition API
- D3.js (可视化交互)
- mitt (事件总线)
- CSS3 动画
